For detailed instructions/screencasts, go to http://www.github.com/amirrajan/specwatchr

1. Open up Ruby command prompt (v 1.9.2)

2. enter the following command to install watchr gem (if you dont have it installed): 
   gem install watchr
   
3. navigate to your solution directory

4. enter the following command to start SpecWatchr:
   watchr dotnet.watchr.rb

5. save a file in your solution and tests should automatically run
